5. font-style

  • italic 기울임
  • oblique 기울임

6. color

텍스트의 색상을 지정 할때 사용
    빨강글자
  • 색상명으로 지정
    • 검정바탕에 흰글자
  • 16진수 값으로 지정
  • rgb로 설정
  • rgba로 설정

7. text-decoration

텍스트에 줄을 그을때 사용(위, 아래, 중간)
  • none
  • underline
  • overline
  • line-throught
    • 링크 스타일일 변경 밑줄 없애기 글자색상 : #666 글자크기 : 1.5em
    test링크

8. text-shadow

텍스트에 그림자 효과를 추가
text-shadow : 가로거리 세로거리 번짐정도 색상 양수 글자의 오른쪽, 아래쪽 음수 글자의 왼쪽, 윈쪽 번짐정도(생략가능) : 양수값 그림자가 모든 방향으로 퍼져 나가므로 크게 표현
HTML5 CSS JAVA SCRIPT
